### usage

#### ios
```shell
npm run ios
```
open option `command + D`
reload `command + R`

#### android
```shell
emulator -list-avds
emulator -avd **** -dns-server 8.8.8.8
npm run android
```
open option `command + M`
reload `R + R`

#### android Actual
```shell
expo start --tunnel
```


#### offline
```shell
expo start --offline
```
